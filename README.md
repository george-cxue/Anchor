# Anchor - Negotiation Strategy Dashboard

A web application created to help negotiators prepare for negotiations through structured analysis, strategic planning, and evidence-based decision-making.

## Purpose

Created for my Negotiations (OIDD 2910) Final Project. Negotiation outcomes are often determined before parties even sit down at the table. **Anchor** transforms negotiation theory into actionable preparation through an interactive, user-friendly platform.

- Quantify your position through BATNA and EV calculations
- Set boundaries with reservation prices
- Understand your counterpart through structured profiling
- Prepare strategic responses using if/then scripting
- Reflect and continuously improve

## Features

### 1. Internal Analysis

- BATNA Calculator
- Reservation Price
- Issues & Positions

### 2. Counterpart Analysis

- Structured Profile
- ZOPA Analysis

### 3. Strategy Tools

- Expected Value Calculator
- If/Then Scripts

### 4. Battle Card

- Print-ready quick reference sheet with walk-away number, opening anchor, prioritized trades, and response scripts

### 5. Reflection

- Personal reflection space with guided prompts
- Community wisdom sharing with example advice from other negotiators

## Technical Stack

- Next.js 15 (App Router), TypeScript, React 19, Tailwind CSS 4
- Type-safe architecture with comprehensive validation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to use the application.

### Usage Flow

1. Define BATNAs and reservation price
2. Analyze your counterpart
3. Model scenarios and prepare responses
4. Generate battle card for negotiation day
5. Reflect and share insights afterward
